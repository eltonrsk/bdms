# BDMS

A blood donation management system built with React, TypeScript, Tailwind CSS, and Supabase.

## Features
- Donor, hospital, and admin dashboards
- Appointment scheduling and donation requests
- Inventory and transfer management
- Authentication and role-based access

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- Supabase

## Getting Started
1. Install dependencies
   ```bash
   npm install
   ```
2. Start the development server
   ```bash
   npm run dev
   ```
